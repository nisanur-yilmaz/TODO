# Todo Web Uygulaması

C#, JavaScript, HTML, CSS ve SQL kullanılarak geliştirilmiş, responsive (mobil ve masaüstü uyumlu) bir Todo uygulamasıdır. Kullanıcılar kayıt olabilir, giriş yapabilir, şifrelerini değiştirebilir ve görevlerini üç kategori halinde yönetebilir: **Yapılacak**, **Yapılıyor**, **Bitti**.

## Özellikler
- Kullanıcı kaydı, giriş ve şifre değiştirme.
- Her kullanıcı için kişisel Todo listesi.
- Görev durumu yönetimi: 
  - `0` — Yapılacak
  - `1` — Yapılıyor
  - `2` — Bitti
- Görev ekleme, düzenleme ve silme (CRUD).
- Responsive tasarım: mobil ve masaüstü uyumlu.

## Teknoloji Yığını
- Backend: ASP.NET Core (C#)
- Frontend: HTML, CSS, JavaScript
- Veritabanı: SQL (SQLite)
- ORM: Entity Framework Core (isteğe bağlı)
